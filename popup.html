⚠️ Security Disclaimer

Please accept this disclaimer to enable the AutoMFA extension.

Bypassing MFA is a security risk. Multifactor Authentication (MFA) protects your account against password theft. Automating or localizing these credentials lowers that barrier.

User Responsibility: You take full, exclusive responsibility for all security risks, data loss, policy violations, or credential issues arising from your use of this extension.

No keys are sent online. All secrets are stored exclusively on your device inside standard local storage.

🔑 Step 1: Get Secret Key

Log in and retrieve your key from Microsoft Security Setup:

  • Go to mysignins.microsoft.com/security-info and log in with your organization (e.g. Purdue) email.
  • Click Add sign-in method and select Microsoft Authenticator.
  • On the prompt, click the link that says "Set Up A Different Authentication App" (or "I want to use a different authenticator app").
  • Press Next until you see the QR Code, then click "Can't scan the QR code".
  • Copy the revealed Secret Key (a string of lowercase letters and numbers).

AutoMFA Config

Paste the secret key you just retrieved from the Microsoft setup screen.

⚠️ Invalid base32 secret. It must consist only of letters and numbers 2-7.

💡 Secrets are case-insensitive and ignores spaces. Standard format uses letters and numbers 2-7.
⚠️ Important: Once setup is complete, you must copy the 6-digit verification code generated by this extension and paste it back into the Microsoft sign-ins page to confirm and finish adding your new device!

⚡ Step 3: Auto-Login Magic

Your secret is saved! Here is how to use the automation features:

💻 Fully Automated Login:
When you log in, the extension automatically clicks through Microsoft's Authenticator screens:
1. Clicks "I can't use my Authenticator app right now"
2. Clicks "Use a verification code"
3. Generates, pastes, and submits the code instantly!
🖱️ Right-Click Paste Options:
Right-click on any input box or login page to see options:
- AutoMFA: Autofill Microsoft Login (forces auto-bypass)
- AutoMFA: Paste into selected textbox (pastes code instantly)
